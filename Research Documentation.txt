PhD Research: Domain-Specific LLM for Cyber Threat Hunting
Project Overview
This research project develops a domain-specific Large Language Model (LLM) for cyber threat hunting, leveraging LLAMA-3.1 8B and MITRE ATT&CK TTPs (Tactics, Techniques, and Procedures).

Module Descriptions
1. Data Preprocessing (01_data_preprocessing.py)
•	Input: MITRE ATT&CK v18.0 Enterprise data
•	Output: Structured instruction-response pairs in JSON format
•	Split: 80% training, 20% testing (stratified by tactic)
2. Synthetic Data Generation (02_synthetic_data_generation.py)
•	Methodology: Evol-Instruct (adapted from WizardLM)
•	Expansion Factor: 5x original training data
•	Strategies:
–	Stylistic variations (40%)
–	Contextual augmentation (30%)
–	Parameter variation (20%)
–	Technique combination (10%)
3. Custom Reward Function (03_reward_function.py)
Mathematical formulation:
R_total = α₁·R_relevance + α₂·R_novelty + α₃·R_impact + α₄·R_feasibility

Where:
- α₁ = 0.30 (Relevance weight)
- α₂ = 0.25 (Novelty weight)
- α₃ = 0.30 (Impact weight)
- α₄ = 0.15 (Feasibility weight)

Target scores:
- Relevance: >0.8
- Novelty: >0.7
- Impact: >0.6
- Feasibility: >0.6
- Overall: >0.75
4. Adversarial Perturbation (04_adversarial_perturbations.py)
•	FGSM: Fast Gradient Sign Method for text perturbation
•	PGD: Projected Gradient Descent for iterative attacks
•	Backtranslation: Semantic-preserving paraphrase generation
•	Robustness Penalty: λ × std_dev(perturbed_scores)
5. Model Training (05_model_training.py)
•	Base Model: Llama 3.1 8B Instruct
•	Quantization: QLoRA (4-bit with LoRA rank 64-128)
•	Optimizer: GRPO (Group Relative Policy Optimization)
•	Curriculum Learning: Perturbation ratio 5% → 15%
•	Dynamic λ Adaptation: Double if ASR > 10%, cap at 0.3
6. Evaluation (06_evaluation.py)
Comprehensive evaluation metrics: - Relevance Score (MITRE alignment) - Novelty Score (compared to known TTPs) - Impact Score (security severity) - Feasibility Score (operational practicality) - Attack Success Rate (ASR) - Statistical significance testing
